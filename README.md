# Transporter Travel

Transporter Travel is a modern website for a travel agency, built with Next.js. It provides users with an intuitive and seamless experience for exploring travel destinations and booking trips.

## Features

- Responsive and dynamic UI
- Fast and efficient with Next.js optimizations
- SEO-friendly pages
- Easy navigation for users

## Installation

To set up the project locally, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/vokx7/transporter-travel.git
   ```
2. Navigate to the project directory:
   ```sh
   cd transporter-travel
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- **Next.js** – React framework for SSR and static site generation
- **React** – Frontend JavaScript library
- **Tailwind CSS** – Utility-first CSS framework for styling
- **Node.js & npm** – Package manager and runtime environment
